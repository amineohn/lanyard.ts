# Lanyard.ts

Lanyard.ts is a TypeScript implementation of [Lanyard](https://github.com/Phineas/lanyard), originally created by
Phineas.


## API Reference

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `id` | `string` | **Required**. Your Discord ID |

#### Get presence by ID

```http
  GET /api/v1/users/${id}
```
